# mancino-signature-assets

Immagini ospitate per la firma email di **Giancarlo Mancino**.

Struttura versionata: aggiornamenti futuri vanno in `/v2/`, `/v3/` senza toccare le cartelle precedenti, così le firme già installate non si rompono.
